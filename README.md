# Login and Registration API (Spring Boot)

A backend service that provides basic user registration and login functionality using Spring Boot.

# Summary
This project demonstrates authentication-related backend APIs and focuses on request handling, data persistence, and clean service layering.

# Key Features
- User registration endpoint
- User login endpoint
- Basic request validation

# Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- H2 Database
- Maven

# Running Locally
```bash
git clone https://github.com/Vidhyasridharan/loginandregistration.git
cd loginandregistration
./mvnw spring-boot:run
